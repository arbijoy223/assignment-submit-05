Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent:
1. innerHtml = Returns or sets the HTML inside an element (includes tags).
2. innText = Returns or sets only the visible text inside an element. It ignores hidden text.
3. textContent = Returns or sets all text inside an element (including hidden text).

Question-3: What is event delegation in the DOM?

Event Delegation:
Event delegation is a technique where you attach an event listener to a parent element instead of individual child elements. This helps when handling dynamic content.

Question-4: What is event bubbling in the DOM?

Event Bubbling:
Event bubbling is when an event on a child element bubbles up to its parent elements.

Clicking the button will show: 1️. "Button clicked!"
2️. "Parent clicked!" (because of bubbling)
To stop bubbling, use event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

Create, Add and remove elements:

Create an element:
let newElement = document.createElement("p");
newElement.innerText = "Hello World!";
Add an element to the DOM:
document.body.appendChild(newElement);
Remove an element:
newElement.remove();
